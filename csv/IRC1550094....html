Department Description

Solaris File Protocols and Interoperability is responsible for Solaris file sharing protocols including NFS and SMB/CIFS and the Lustre high-performance, scalable clustered storage file system.  This development and test is leading innovation in shared, network file systems and interoperability between diverse platforms, protocols, and interconnects.  We contribute technology to Solaris as well as into Oracle's Open Storage systems including the ZFS Storage Appliance. Our focus and passion is on delivering robust, enterprise-ready customer solutions and providing technology leadership in standards-based file protocols.

Additional Details

This is a position for a senior software engineer who will participate on a core team developing support for the Microsoft SMB 2 file sharing protocol.  This project will build on the existing Solaris SMB/CIFS service, a mature and robust implementation of Windows File Sharing which is a central feature of the ZFS Storage Appliance.  The Solaris SMB implementation is noteworthy in its unparalleled level of integration with the ZFS file system and NFS file sharing protocols  offering a unified, cross-protocol name space, a single, unified ACL-based security model, seamless cross-platform identity support, and cross-protocol file locking.

This position requires 8+ years of software development experience in the C/Solaris/Unix/Linux environments.  Ideally, the candidate would have design/development experience in file systems or file sharing protocols such as CIFS/SMB or NFS. Since this position involves developing Microsoft interoperability capabilities for Solaris familiarity with Windows file protocols and naming and security services (ex. Active Directory, LDAP, Kerberos) is quite desirable.

The candidate is expected to demonstrate solid software engineering practices: complete and thoroughly considered designs, mature coding patterns, and methodical debugging techniques.  To succeed in this role, you'll need a strong systems programming background, a knack for debugging and troubleshooting, and the ability to thoroughly learn new technology areas.
