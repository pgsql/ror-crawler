Detailed Description
Analyze engineering, business, and all other data processing problems for application to electronic data processing systems. Analyze user requir ements, procedures, and problems to automate or improve existing systems and review computer system capabilities, workflow, and scheduling limitations. Specifically, under limited supervision, will develop and implement innovative software solutions to solve complex technical and business issues, providing user requir ements definition, and transforming functional software requir ements into detailed designs. Will prepare, write, and develop detailed relational database designs to be used in the implementation of the Oracle software applications. Responsible for designing, developing, testing, and delivering software objects to extend and customize Oracle packaged applications to meet customer requir ements. Also, develop software to accomplish data conversion and interfaces to and from other systems; manage technical aspects of multiple strategic studies and undertake large scale, complex strategy and design efforts; and participate in the preparation of project proposals, develop system and user documentation, and train end-users.
