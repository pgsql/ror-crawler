Department Description
The database security group is chartered to provide security infrastructure components for Oracle product groups and develop security products/solutions for Oracle customers. Oracle is the leader and pioneer in database security. Our group continues to innovate and extend the lead on our competitors. We develop core database security functionality, network security and solutions that address enterprise-wide security challenges, especially by consolidating and centralizing security relevant information and policies, thereby enabling better manageability and stricter enforcement. Homeland security, Privacy, Compliance, Application Hosting, Defense, Finance, and Healthcare are among the industries that benefit from this technology. Database Security is HOT! Security is a essential for businesses and consumers in the information age. Interested in being part of a group building new cutting-edge revenue generating products? This is the team to join!

The team is responsible for building the Oracle Database Firewall appliance. Oracle Database Firewall is the first line of defense for databases, providing real-time monitoring of database activity on the network. Highly accurate SQL grammar-based technology blocks unauthorized transactions, helping prevent internal and external attacks from reaching the database.
