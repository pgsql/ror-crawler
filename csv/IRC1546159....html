Department Description
The Java Platform Group is looking for an experienced, passionate, and highly motivated Software Quality Engineer to join our world class development efforts. Our team is responsible for delivering the Java platform that is used by millions of developers. We are looking for a seasoned engineer with a thorough understanding of software security, testing methodologies, a passion for quality, a strong technical background to advance our processes, and a desire to make a positive change in our product.

Responsibilities:

We hire developers to write code to test code. Our goal is to have engineers writing robust, reliable, repeatable and maintainable tests that find issues early and cover the area of the component under testing. Our quality engineers are in the product source code as much as they are working with test source code and are building the framework used for testing. Growth in this role means growing as an engineer: strong designs that are built to last, deep component knowledge and broad scenario experience will be measures of success as a candidate grows in this role. The primary focus for this position will be on server technologies: Core Java, libraries, security

• A strong technical background as well as good verbal and written communication skills.
• Candidates with previous test automation experience (junit, testNG) will be given priority
• You will be expected to work with a wide range of features and component teams in order to drive quality in security, networking, core library, tools, web services
• Relevant experience in Java Cryptography (JCE), Java Secure Socket Extension (JSSE), PKI, PKCS, Kerberos, SSL, SMIME, HTTPS would be desirable
• Create the next generation of testing tools, techniques, and infrastructure to increase the speed of release cycles
• Work closely with engineering teams to understand new technologies, design and develop functional test suite to ensure the quality of the product
• Designing and developing test harnesses and test applications.
• Writing test plans and test cases to ensure product quality, increase code coverage and minimize validation escapes to customers.
• Ensuring timely software releases by improving efficiency, productivity and consistency through test automation
