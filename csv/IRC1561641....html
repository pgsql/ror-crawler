ESSENTIAL FUNCTIONS

  • Research and analyze a customer’s existing license set using Oracle’s applications. Determine license ownership by verifying ordering documents and comparing them to Oracle’s records online.
  • Determine the correct migration (conversion from one older program and license metric to a new program and license metric) by utilizing Oracle’s price list.
  • Properly apply Oracle migration policies to determine the user minimums and maximums of the new license configuration. Use the current price list to determine the list license fees for the new configuration.
  • Determine the customer’s existing support stream by reviewing the the current support fees paid; validate fees with Oracle’s service contracts database. Verify all paid invoices and perform additional mathematical calculations if necessary. 
  • Compile and consolidate necessary data from the research performed. Enter data into a complex Microsoft Excel template and alter the template as needed including applicable discounts and using the government discount schedule for government customers). Ensure accuracy of the migration calculation results.
  • Send completed migration template to the Sales Representative, Support organization, and Contracts teams. Provide follow-up if necessary.
  • Respond to questions and inquiries regarding completed migrations. Lend advice to sales representatives via thorough explanations of the migration process and outcome. Assist the sales team by suggesting alternative migration paths and scenarios that would be most beneficial to the customer.
  • Work closely with the contracts organization when migration reaches the booking process. Assist the contract specialist by providing explanation and methodologies that were applied to the migration.
