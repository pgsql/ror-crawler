Brief Description
Analyzes software for globalization specifications. Documents and communicates the results. Designs software globalization solutions in the form of globalization design documents.
Detailed Description
Designs quality test measures for a broad range of software/hardware products. Prepares installation scripts and local project files for submission to Language/Localization Specialists. Integrates programs, such as (but not limited to) Windows NT, Internet Explorer, and SQL Server in localized languages. Modifies processes to handle building various languages and installers to support international languages. Designs and implements test plans and test cases. Demonstrating knowledge of developmental and localization tools, methodologies and languages, communicates with Language/Localization Specialists and provides support on context for translation strings in addition to training instructions and support for new Language/Localization Specialists.
