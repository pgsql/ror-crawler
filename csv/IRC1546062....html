 About Us and the Product

 

At Oracle, we are defining the future of the complete, simple and unified software and hardware platform.  A key part of Oracle’s overall development strategy, Oracle Application Development Framework (Oracle ADF) is a unique Java EE framework that makes enterprise Java and service oriented development accessible to the broader application development community.  Oracle ADF is already used by many large corporations and is the cornerstone of the Oracle Fusion Architecture.

 

As part of the ADF product suite, ADF Mobile extends Oracle Application Development Framework to mobile users. Using Oracle JDeveloper, application developers can rapidly develop mobile applications that support mobile users accessing critical business data through either on-device mobile client or mobile. This team is responsible for architecting and developing the software infrastructure to assist development of enterprise applications leveraging Oracle technologies. 

 

About the Role

 

This position is located at our headquarters location in Redwood Shores, CA (SF Bay). We are seeking a hands-on Development Engineer with advanced SCM development experience and Java/J2EE Development skills.  The ideal candidate has an extensive background in commercial enterprise application and/or tool development, as well as experience in Source Code Management tools.  Candidates will be responsible for:

  • Integrating state of the SCM solutions (GIT,  SVN, Perforce, ...)
  • Designing and developing Code Review other tools related to Application Lifecycle Management
  • Designing and developing advanced Source Code and Source History visualization tools
  • Staying current with state of the art SCM and other ALM related tools and technologies
