Detailed Description
Develop, apply and maintain quality standards for company products with adherence to both internal and external standards. Develop and execute software test plans. Analyze and write test standards and procedures and maintain documentation of test results. Analyze test results and recommend corrective actions. Develop, maintain, and review functional test scripts according to provided functional and technical designs. Run functional test scripts and document test results. Perform bug fix testing by running a functional test script or creating ad hoc test cases for affected functionality. Develop and maintain automated test scripts from provided functional and technical designs, functional test scripts or ad hoc for bug fix testing. Run automated test scripts and document test results. Participate in support of automated testing suite for use by groups within Oracle. Participate in defect and issue reviews as well as resolution process. Participate in testing project management by contributing task definitions and estimates for testing project plans and preparing progress reports and metrics for Project Leader and Quality Assurance management team. Participate in the definition and implementation of Quality Assurance processes and standards and contribute to the definition, validation, and implementation of Quality Assurance testing tools.
Job Requirements
Employer will accept Bachelor’s degree in Computer Science, Engineering or related technical field and 3 years of work experience in job offered or 3 years of work experience in a computer-related occupation. Experience to include: 1) Testing / QA retail applications; 2) utilizing Oracle Retail Merchandising System, Oracle Retail Sales Audit and Oracle Retail Invoice Match; 3) SQL, PL/SQL and Java; 4) designing and developing test cases and manual and automated applications; 5) implementing automated test suites, analyzing tests, resolving errors and implementing solutions; 6) performing functionality and regression testing; Experience may be gained concurrently. Any suitable combination of education, training, or experience is acceptable. Location: Minneapolis, MN
