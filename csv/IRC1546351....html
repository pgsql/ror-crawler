Department Description
As part of the Desktop Virtualization Group, the Sun Ray Software Engineering Team is responsible for Oracle's Sun Ray solution. The solution comprises the stateless Sun Ray Client device and the Sun Ray Software infrastructure creating a highly secure, centrally managed, globally accessible virtual desktop infrastructure for use with Windows, Linux and Solaris desktops.  The department is responsible for the client firmware, network protocols, and server management components.  Oracle's virtualization solutions are developed specifically to make enterprise software and hardware dramatically easier to deploy, access, manage, and support, in order to shorten deployment and provisioning times, while reducing the cost of operations and optimizing performance and availability.
