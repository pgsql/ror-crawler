Job Title Applications Developer 2

Organization Name Fusion Applications Tool Kit Development
Department Description

Applications Tool Kit (ATK) is responsible for building central components in Fusion Applications. A team of enthusiastic folks who understand the blend of technology and applications like

  • Fusion Home Page
  • Help Portal
  • Reporting Pane and Workarea
  • Watchlist and Popup Notifications
  • ESS Custom Registration
  • UMS based Messaging, etc to name a few

ATK is also responsible for some of the web 2.0 features in Fusion Applications like Contextual Actions, Tagging , Community Star Ratings to name a few.

We interface closely with both application teams helping them adopt certain pieces of technology and with the technology team to bubble up the needs.

Brief Description
Analyze, design develop, troubleshoot and debug software programs for commercial or end user applications. Writes code, completes programming and performs testing and debugging of applications.
Detailed Description
As a member of the software engineering division, you will perform detailed design based on provided high level design specifications. Assist in system planning, scheduling and implementation. Build enhancements (including new product features) and resolve bugs. Build and execute unit test and unit test plans. Review integration and regression test plans created by QA. Interact with QA and porting engineering about problems in the code.
Job Requirements
Duties and tasks are standard with some variation; displays understanding of roles, processes and procedures. Performs moderately complex problem solving with assistance and guidance in understanding and applying company policies and processes. Preferred Qualifications: BS degree or equivalent experience relevant to functional area. 1 year of software engineering or related experience.
Additional Details

The ideal candidate for this position will possess:

  • Bachelors or Masters Degree in Computer Science.
  • 2 - 4 years of experience in product development using Java (J2EE), Oracle, SQL.
  • Ability to work under pressure.
  • Should be a fast learner and be able to collaborate with senior members / architects in the team.
  • Excellent communication skills.
  • Familiarity with Oracle Fusion ADF, SOA Suite, Web 2.0 features is a big plus, but not mandatory
